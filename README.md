# Aquarium Wallet

A comprehensive mobile cryptocurrency wallet application built with React Native.

## Features

- Multi-chain wallet support (Ethereum, Polygon, Arbitrum, Optimism, Base, BSC)
- NFT management and trading
- Token swapping
- WalletConnect integration
- Biometric security
- Transaction management
- Settings and preferences

## Project Structure

The project follows a modular architecture with clear separation of concerns.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- React Native CLI

### Installation

```bash
npm install
```

### Running the App

#### iOS
```bash
npm run ios
```

#### Android
```bash
npm run android
```
